windows下安装Mysql-5.7.37

槿苏 2022/7/24 mysql

记录 mysql-5.7.37 的安装包安装过程

# 1、下载 mysql-5.7.37 安装包

下载地址 (opens new window)

# 2、解压缩 mysql-5.7.37 安装包到指定目录

以 D:\Program Files\mysql-5.7.37 安装目录为例

解压后的目录结构

解压完是没有 data 目录和 my.ini 文件的

这里在 mysql-5.7.37 安装目录下新建一个 my.ini 文件，写入以下内容并保存

[client]
port=3306
default-character-set=utf8
[mysqld]
# 设置为自己MYSQL的安装目录
basedir=D:\Program Files\mysql-5.7.37
# 设置为MYSQL的数据目录
datadir=D:\Program Files\mysql-5.7.37\data\
port=3306
character_set_server=utf8

# 3、准备安装

#1、管理员运行cmd进入mysql-5.7.37的bin目录，对mysql进行初始化
	#初始化结束会生成data的文件夹，其中保存系统和用户的数据
	#方式一：创建一个"root"@"localhost"账户并设置空密码
		#root为超级管理员账号，没有密码不安全，创建后需要自己手动设置密码
		mysqld --initialize-insecure --console

	#方式二：创建一个"root"@"localhost"账户并设置随机密码，执行完毕后会在控制台中打印出生成的密码
		#但是密码比较复杂难以记住，建议先使用方式一，执行完手动设置密码。
		#也可以使用此方式，后续手动更改密码
		mysqld --initialize --console
#2、安装为系统服务
	#出现Service successfully installed.表示服务安装成功
	mysqld -install mysql-5.7.37
#3、将D:\Program Files\mysql-5.7.37\bin(实际安装位置的bin目录)添加到Path环境变量
#4、常用命令
	mysld --install 服务名				#安装自定义名称的MySQL服务
	mysqld -install					  #安装默认名称为MySQL的服务
	mysqld -remove                    #移除MySQL服务
	mysqladmin -u 用户名 -p password   #设置密码
	mysql --version                   #查看MySQL版本
	net start 服务名  					#启动MySQL服务
	net stop 服务名  				    #断开MySQL服务
	sc delete 服务名   				#删除MySQL服务
